5. V5 Classification Engine

Rule baseline → layer profiles → graph topology → DeepSeek on low-confidence entities. Column/Wall classify for exclusions only.

flowchart TD
    E[Entity] --> R{Rule baseline}
    R -->|Hard layer| L1["Locked type\nOpening · Column · Beam"]
    R -->|Soft layer| L2["Geometry heuristics\nbeam line · column · wall"]
    R -->|Teach rule| L3["Layer profile\nfrom Tagged Files_2 corpus"]
    L1 --> G[Structural graph topology]
    L2 --> G
    L3 --> G
    G --> C[V5 confidence score]
    C --> D{Below 75%?}
    D -->|Yes| DS["DeepSeek + RAG\natlas · KB · neighbours"]
    D -->|No| F[Final type]
    DS --> F

    F --> QTY{Quantity phase?}
    QTY -->|Slab · Beam| CALC[Concrete + shuttering]
    QTY -->|Column · Wall · Opening| EXCL[Exclusions + context only]
