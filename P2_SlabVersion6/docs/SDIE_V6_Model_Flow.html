Eleven numbered steps from DXF load through export.

flowchart TD
    START([run_pipeline.py]) --> S1

    S1["① Load DXF\nunits · scale · extents"]
    S2["② Auto-discover layers\nframe · annotation · void · column"]
    S3["③ Resolve floor zone\nTHK label clusters"]
    S4["④ Load teach artifacts\nAtlas + KB · GENERIC = all projects"]
    S5["⑤ Extract entities\nstructural · cutout · hatch void · columns"]
    S6["⑥ Classify — V5 engine"]

    S8["⑧ Build structural graph\nnodes + topology edges"]
    S9["⑨ Slab detection\nexclusions → beam grid → bays"]

    S6 --> S7 --> S8 --> S9 --> S10 --> S11

4. V5 Classification Engine

Rule baseline → graph topology → DeepSeek on low-confidence entities.

flowchart TD
    E[Entity] --> R{Rule baseline}
    R -->|Hard layer| L1["Locked type\nOpening · Column · Beam"]
    R -->|Soft layer| L2["Geometry heuristics\nbeam line · column · wall"]
    R -->|Teach rule| L3["Best layer profile\ncross-project if GENERIC"]
    L1 --> G[Structural graph topology]
    L2 --> G
    L3 --> G
    G --> C[V5 confidence score]
    C --> D{Below 75%?}
    D -->|Yes| DS["DeepSeek + RAG\natlas · KB · neighbours"]
    D -->|No| F[Final type]
    DS --> F

5. Quantity Engines

Beam takeoff (length × section) and slab takeoff (area × THK) combined into totals.

flowchart LR
    subgraph BEAM["Beam takeoff"]
        B1[Classified Beam lines] --> B2[Centerline length]
        B2 --> B3["Section WxD\ntags or drawing default"]
        B3 --> B4["Concrete + shuttering m³/m²"]
    end

    subgraph SLAB["Slab takeoff"]
        SL1[Classified framing] --> SL2[Exclusions applied]
        SL2 --> SL3[Closed bays via beam grid]
        SL3 --> SL4[Nearest THK label]
        SL4 --> SL5["Area × thickness → m³"]
    end

    B4 --> T["Combined totals\nslab + beam concrete"]
    SL5 --> T
